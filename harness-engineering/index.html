Harness Engineering

Master the production engineering of AI agent harnesses: context budgeting, sandbox infrastructure, permission systems, model routing, observability, and scaling. Build the software layer that powers products like Claude Code, Cursor, and Devin.

Course Progress 0% Complete

Course Philosophy

  • Production-first - Every concept maps to real shipping decisions at Claude Code, Cursor, or Devin scale
  • Architecture over abstractions - Understand the engineering, not just the framework APIs
  • WHY before HOW - Know why a harness uses token budgeting before learning the algorithm
  • Build to learn - Each module includes a mini-project building toward a complete harness

Who This Course Is For

  • Senior/staff engineers building AI-powered developer tools or agents
  • Engineers preparing for interviews at companies building LLM harnesses (Anthropic, Cursor, Cognition, etc.)
  • Backend engineers who understand system design and want to specialize in AI infrastructure
  • Prerequisite: Complete Gen AI modules 1-12 or equivalent LLM/agent knowledge

Prerequisites

Required Knowledge

  • LLM fundamentals - Tokenization, attention, generation parameters (Gen AI Modules 1-7)
  • Agent basics - ReAct pattern, tool calling, function calling (Gen AI Module 10)
  • System design - Distributed systems, queues, caching, load balancing
  • Python + TypeScript - Comfortable reading both; async/await patterns

Helpful but Not Required

  • Docker / container orchestration experience
  • OpenTelemetry / observability stack experience
  • Production experience with LLM APIs (OpenAI, Anthropic)
